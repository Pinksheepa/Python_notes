# Python基础

## 数据类型
| 数据类型 |                                    说明                                    |
| :------: | :------------------------------------------------------------------------: |
|   整数   |                           -1，0xffff，1_000_000                            |
|  浮点数  |                              0.1，2e-1（0.2）                              |
|  字符串  | 'xy'(表示x，y)，转义符号\ <br> Python允许用r''表示''内部的字符串默认不转义 |
|  布尔值  |                     True、False，可and \ or \ not运算                      |
|   空值   |                                    None                                    |
| **变量** |                             **用“=”动态赋值**                              |
|   常量   |                          通常大写表示，本质为变量                          |
